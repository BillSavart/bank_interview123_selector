# 公股銀行新手村 Logo 設計稿

這組設計先只放在 `docs/brand/`，沒有接到網站畫面。

## 檔案

- `logo-primary.svg`：橫式主 logo，適合 navbar、OG 圖、簡報和文件。
- `logo-mark.svg`：純圖標，適合 favicon、app icon、社群頭像。
- `logo-design-board.svg`：完整展示板，包含主 logo、深色底、小尺寸、單色版與色票。

## 設計方向

- 村口屋頂：呼應「新手村」，但線條保持成熟，不走太遊戲化。
- 題卡：代表面試題庫、試場資訊與準備清單。
- 指南針：代表幫考生找準備方向，不讓品牌看起來像官方銀行。
- 深藍綠搭配金色：延續網站現有的可信工具感，金色只作為重點提示。
